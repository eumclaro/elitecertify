# Regras de Trabalho — Antigravity

Seguir estas regras em **todos** os prompts deste projeto:

1. **Instruções numeradas e objetivas** — sem ambiguidade no que deve ser feito
2. **Escopo cirúrgico** — nunca alterar mais arquivos ou linhas do que o estritamente necessário para a tarefa
3. **Confirmação visual antes de avançar** — sempre solicitar screenshot ou validação no DevTools antes do próximo passo
4. **Build de produção após cada mudança relevante** — confirmar que não há erros de compilação
5. **Backup antes de alterações destrutivas** — criar `.bak` ou fazer git commit antes de deletar ou sobrescrever arquivos importantes
